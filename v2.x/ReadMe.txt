Installation Notes
==================

1. Create the Database from Database\Create.sql or Restore a blank from Web\App_Data or leave it if you are running sql express.
2. Run the Database\Data.sql to populate some category/tag.
3. Create an account with the following external services:
	-http://www.pageglimpse.com/
	-http://www.websnapr.com/
	-http://antispam.typepad.com/
	-http://akismet.com/
	-http://defensio.com/
	-http://recaptcha.net/
4. Open The web.config in your prefered text editor.
5. Try find with "YOUR-" and replace with your value.
6. Open the Kigg.sln and run.
